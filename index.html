Welcome to Computer Vision Image Classifier API!

Introduction

This site represents a simple project with a few learning objectives, including:
  • Learn the basics of image classification using computer vision with Pytorch
  • Create a lightweight web application using Flask
  • Deploy an application to the web using Heroku

How to Use the API

  1. Choose an image that you would like to classify, and save it as a JPEG file
  2. Next, either:
    • Navigate to the predict page to upload your file to the API, or
    • Send a POST request to this API at the endpoint '/predict' with the JPEG in the body of the request, using the key 'file'
  3. You will receive back a JSON object including the model's top 5 class predictions. For each prediction, you will receive the class ID, class name, and the model's confidence level in the prediction.

Additional Comments

The API uses a pretrained Pytorch model (googlenet) that was trained on ImageNet. No additional training has been done. As a result, the model does not always work very well at classifying images (maybe that was something I should have said at the top!). This is the reason that I decided to return the top 5 predictions from the model, instead of only the top prediction, along with the confidence level. Obviously, if you see 5 predictions that don't seem like related classes, each with relatively similar confidence levels, the model didn't work very well. Sorry :(

As a future project, I hope to narrow the focus of the API to a particular subset of image classes (e.g. zoo animals or food items), and use Transfer Learning to enhance the model's accuracy. But, as much as I'd like to dive into that now!...it is outside of the scope of what I set out to do at this time.
